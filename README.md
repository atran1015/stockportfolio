# How to run this program (Ubuntu)
```
download the zip file or do a git clone
cd stockportfolio/
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt

```
# How to run individual files
```
chmod u+x [filename].py
./[filename].py

```
